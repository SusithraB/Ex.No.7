# Aim: 
To develop a prompt-based application using ChatGPT - To demonstrate how to create a prompt-based application to organize daily tasks, showing the progression from simple to more advanced prompt designs and their corresponding outputs.



## **AI Tools Required**

* ChatGPT (Primary LLM for prompt design and response generation)

---

# **Introduction**

Prompt-based applications use structured instructions (prompts) to guide large language models (LLMs) like ChatGPT to perform specific tasks. In this experiment, a **Personal Productivity Assistant** is designed to manage daily tasks, schedule reminders, provide wellness tips, and respond to general queries. The assistant is improved through progressive prompt engineering—from simple prompts to advanced contextual prompts.

---

# **Procedure**

1. Define the core features of a personal productivity assistant.
2. Design multiple prompts with increasing complexity.
3. Test prompts using ChatGPT and observe outputs.
4. Compare simple vs advanced prompt responses.
5. Simulate user interaction using natural language inputs.
6. (Optional) Include memory-like behavior using contextual prompting.

---

# **Prompt Design and Outputs**

---

## **1. Simple Prompt**

### **Prompt 1**

Help me manage my daily tasks.

### **Output**

* You should create a to-do list.
* Suggest organizing tasks by priority.
* Remind about pending work.

### **Observation**

* Very generic response
* No structure or personalization

---

## **2. Intermediate Prompt**

### **Prompt 2**

"Act as a personal assistant. Organize my tasks for today:

* Finish assignment
* Call mom at 6 PM
* Study Python for 2 hours
  Also suggest a schedule."

### **Output**

**Daily Schedule:**

* 9:00 AM – 11:00 AM → Finish assignment
* 11:30 AM – 1:30 PM → Study Python
* 6:00 PM → Call mom
* Evening → Review pending tasks

### **Observation**

* Better structure
* Time allocation introduced
* Still no adaptability

---

## **3. Advanced Prompt (Final Application Design)**

### **Prompt 3**

Design a personal productivity assistant that helps manage daily tasks, schedule reminders, suggest wellness tips, and adapt based on user preferences. It should interact in natural language and behave like a smart AI assistant.

---

### **Output (LLM Response)**

## **Personal Productivity Assistant Features**

### **1. Daily Task Manager**

* Accepts natural language inputs like:

  * “Remind me to submit assignment by 5 PM”
* Automatically categorizes tasks:

  * Academic
  * Personal
  * Health
* Provides daily summaries:

  * Completed tasks
  * Pending tasks

---

### **2. Smart Scheduler**

* Suggests optimal time slots for tasks
* Detects overlapping schedules
* Adjusts plans based on workload
* Sends reminders for upcoming deadlines

---

### **3. Wellness Assistant**

* Suggests hydration reminders
* Recommends study breaks (Pomodoro-style)
* Encourages sleep and exercise balance
* Example:

  * “You’ve been studying for 2 hours. Take a 10-minute break.”

---

### **4. Adaptive Behavior**

* Learns user preferences:

  * Preferred study time
  * Productivity patterns
* Adjusts scheduling suggestions accordingly

---

### **5. General Query Support**

* Answers questions like:

  * “What is the best way to stay productive?”
  * “How can I reduce stress during exams?”

---

# **Simulated User Interaction Example**

**User:** Remind me to complete my project by tomorrow evening
**Assistant:** Your task has been added. Would you like me to allocate study time today?

**User:** Yes
**Assistant:** Suggested plan:

* 6:00 PM – 8:00 PM → Project work
* Reminder set for 7:45 PM break
